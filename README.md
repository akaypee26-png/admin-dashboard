# 🚀 Admin Dashboard (Node.js + MongoDB)

A full-stack Admin Dashboard built using Node.js, Express, and MongoDB with authentication and user management.

---

## 📌 Features

- 🔐 Admin Signup & Login (JWT Authentication)
- ⏳ Session Timeout with Auto Logout
- 👥 User Management (Add / Edit / Delete)
- 🔎 Search Users
- 📄 Pagination (Next / Prev pages)
- 🔔 Toast Notifications (Success / Error)
- 🎨 Clean Admin Dashboard UI

---

## 🛠 Tech Stack

- Backend: Node.js, Express
- Database: MongoDB (Atlas)
- Authentication: JWT
- Frontend: HTML, CSS, JavaScript
